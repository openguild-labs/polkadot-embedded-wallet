# Polkadot Embedded Wallet

## MPC & Key Sharding

### Processes

- Key generation & sharding
- Signing
- Verification
- Access

### Features

- Multi-user approval
- Key recovery e.g. social recovery
- Modify quorum without changing keys
- Protocol-agnostic
- Key refresh
- Offline signing
